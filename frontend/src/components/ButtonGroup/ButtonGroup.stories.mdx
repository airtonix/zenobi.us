# ButtonGroup

A ButtonGroup holds Buttons in an arrangement to suit spacing within and between its neigbours


<Canvas>
    <Story name="basic">
        <ButtonGroup joined direction={ButtonGroupDirections.Horizontal}>
            <Button label='one' />
            <Button label='two' />
            <Button label='three' />
        </ButtonGroup>
    </Story>
</Canvas>
