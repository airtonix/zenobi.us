The size is a preset to the font size.

| size     | font-size |
| -------- | --------- |
| small    | 12        |
| normal   | 16        |
| big      | 18        |
| large    | 24        |
| huge     | 48        |
| massive  | 64        |
| enormous | 96        |

The size can be `inherited` from the containing element
